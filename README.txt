	Foi utilizado o power BI para apresentação dos dados, caso haja a necessidade
de conexão com o banco, basta conectar localmente pelo "obter dados"


---------*-------- RESPOSTA ---------*--------


Ánalisando os dados é possivel notar que os maiores índices de mortalidade vem de cidades 
"pequenas" onde o numero de casos não é tão alto, devido a isso o numero de óbitos tem um 
efeito maior sobre o índice comparando com outras cidades "maiores".
